DLSR MAPS - versión optimizada

Archivos principales:
- index.html
- acceso.html
- placa.html
- contrato.html
- ubicacion.html
- styles.css
- app-core.js
- data-config.js
- sw.js
- manifest.json
- logo.png
- logo_fondo_blanco.png

Instrucciones:
1. Suba todos estos archivos a la raíz del repositorio, reemplazando los anteriores.
2. Mantenga los archivos Excel de datos también en la raíz del repositorio.
3. Cuando agregue un Excel nuevo, agréguelo a la lista de data-config.js.
4. Después de publicar cambios, recargue con Ctrl + F5 para limpiar caché del navegador.

Cambios aplicados:
- Se quitaron los mensajes visibles con nombre de archivo y cantidad de registros.
- Las búsquedas cargan y procesan el Excel una sola vez por página.
- Se mejoró el diseño visual y la claridad de resultados.
- Se redujeron operaciones repetidas para mejorar velocidad.
- Se conserva búsqueda por placa, contrato, ubicación, enlaces a Maps/Waze y geolocalización.
